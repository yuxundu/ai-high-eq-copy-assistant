# AI高情商文案助手 – 市场定位  
## 产品概述  
AI高情商文案助手是一款轻量级网页工具，方便帮助中文用户快速生成高情商的沛通文字和文案。它利用AI生成温暖、得体、符合语境的短句，适用于日常社交、职场沛通和节日祝福等场景。  
## 目标用户  
- **主要用户：** 活跃在社交媒体和聊天软件的20–35岁中国用户。  
- **次要用户：** 需要高质量沛通的职场人士（如邮件、道歉、请求），以及希望输出精彩评论或标题的在线内容创作者。  
## 市场需求  
许多人在表达自己时感到困难，抱疚提词生硬或尴尬。他们想：  
- 避免生硬或笨扫的表达。  
- 写出让人舒适的回复、祝福或称赞。  
- 节省撰写社交内容或短文案的时间。  
本产品通过提供贴合语境的高情商短句解决这些症点。  
## 关键使用场景  
- **个人关系：** 为朋友或伴侣生成安慰、道歉或感谢的信息。  
- **社交互动：** 为朋友圈、小红书、视频评论等平台生成有趣或真挚的评论和标题。  
- **工作沛通：** 编写礼谊、得体的邮件、请求或进展说明。  
- **节庆活动：** 生成节日祝福、生日祝福等真诚的问候文案。  
## 差异化与价值主张  
- **本地化中文语气：** 定制化的提示词确保输出符合中文表达习惯与文化语境。  
- **多种风格：** 用户可选择谊皮、正式、诗意或幽默等不同风格。  
- **即时结果：** 简单界面快速提供多个选项，减少构思时间。  
- **免费版本 + 高级版本：** 基础版提供有限生成次数，付费升级可解锁无限使用和更多模板。
