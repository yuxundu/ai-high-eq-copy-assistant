# 微信救场场景交互文档  
## 场景概述  
这个场景用于解决用户在微信聊天时突然不知道怎么回复的问题。用户通常在收到让人尴尬、难以回应或需要高情商处理的消息时，通过工具快速生成适合的回复。  
## 用户目标与触发  
- 用户刚收到一条信息，不知道如何回复，担心语气不当。  
- 希望在几秒钟内得到一条可以直接发送的回复。  
- 不想过多解释，希望输入尽量简单。  
## 输入逻辑  
1. **文本输入**：用户手动输入对方说的最后一句话或简单描述情况。  
2. **截图上传**：用户上传聊天截图，系统通过 OCR 识别最后的对话内容。  
3. **关系选择**：提供可选关系标签，如朋友、恋人、同事、领导、家人，帮助调整语气。  
4. **语气选择**：提供温柔、幽默、直接、稳重等选项，指导 AI 生成不同风格。  
## 生成逻辑  
- 系统根据输入内容和语气生成 2–3 条不同风格的回复，每条回复配有语气标签。  
- 每条回复旁边提供“复制”“收藏”按钮以及“生成卡片图”选项。  
- 免费用户可以预览一条，更多回复需付费解锁。  
## 付费触发点  
- 当用户尝试复制或查看完整回复列表时，弹出购买提示。  
- 收费模式为按次支付，解锁一次完整生成（3–5 条回复）。  
- 提供套餐选项如 5 次包，以提升复购率。  
## 分享与输出  
- **文字复制**：快速复制后粘贴到微信对话。  
- **生成图片**：对于愿意在朋友圈或小红书分享的用户，可点击“生成卡片图”，生成带轻水印的心情卡片。  
- **用户提示**：复制后弹出提示，引导用户生成分享图，并说明水印不会影响内容美感。  
## UX 提示语  
- 生成完成后：“已生成 3 条高情商回复，选一条复制发送吧。”  
- 复制成功后：“已复制，要不要生成一张卡片图？”  
- 支付提示：“解锁完整回复 ¥1.9，可立即复制发送。”
